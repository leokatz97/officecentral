# BBI Brand Canonical Map

_Generated by VENDOR-NORMALIZE-1 · 2026-05-12 · Read-only input for VENDOR-NORMALIZE-2 + PE Pass 3 batch prompts._

---

## Purpose

This document establishes the single canonical brand vocabulary for Brant Business Interiors' Shopify catalog. It resolves 50 raw `specs.manufacturer` string variants found across 152 enriched products into 20 canonical brand names, and records which brands are standalone storefront identities versus internal corporate divisions that fold into a parent.

Downstream steps consume this map as a reference vocabulary: VENDOR-NORMALIZE-2 applies it to the 152 already-enriched products; PE Pass 3 batches 3/4/6 use it to populate `vendor_override` per product; COLLECTION-CLEANUP-1 uses the `storefront_callable` flag to decide which brand callouts survive on category pages.

---

## Data sources audited

- **593** active products checked for `specs.manufacturer` metafield

- **441** products unenriched — out of scope for this map; classified during PE Pass 3
- **50** raw string variants identified in the enriched set
- **20** canonical brands resolved (18 from data + Keilhauer + Teknion added as forward references)

## Sub-brand decisions — Global Furniture Group

Global Furniture Group is a publicly traded Canadian conglomerate. The divisions below appear in raw `specs.manufacturer` strings but are **not** standalone storefront brands — their products surface under the "Global Furniture Group" canonical brand at the storefront level.

| Division | Decision | Reasoning |
|---|---|---|
| **Global Upholstery Co.** | Fold into Global Furniture Group | Corporate manufacturing entity behind upholstered seating. Not marketed under this name to buyers. |
| **Global Fileworks** | Fold into Global Furniture Group | Internal filing products division. Products sold as "Global" filing, not "Global Fileworks" filing. |
| **Newland** | Fold into Global Furniture Group | Collaborative/lounge sub-line. Low product count (appears in 3 combo strings only). |
| **Global Basics** | Fold into Global Furniture Group | Budget sub-line. Not separately marketed to BBI's B2B buyers. |
| **OTG / Offices to Go** | **Standalone** | Consumer-recognizable brand. Buyers search "Offices to Go" directly. Highest-volume division in enriched set (54 products). |
| **ObusForme** | **Standalone** | ICP-referenced ergonomic brand with its own buyer recognition. Callable. |
