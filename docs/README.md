# docs/

All project documentation, grouped by purpose. Start at the root [README.md](../README.md) for a one-page orientation.

| Folder | What lives here |
|---|---|
| `plan/` | The active fix plan, status snapshots, and the idea parking lot. Update these as work progresses. |
| `strategy/` | Brand voice, ICP, segmentation, competitor research. Mostly locked — change deliberately. |
| `workflows/` | How-to runbooks for recurring shop operations (shipping, taxonomy, design). |
| `reviews/` | Auto-generated review artifacts written by `scripts/build-*.py`. Regenerate rather than hand-edit. |
