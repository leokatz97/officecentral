## A/B Testing Methodology

### Steps
1. Record baseline CTR for 30 days (minimum 1,000 impressions)
2. State hypothesis: "Adding X will increase CTR by Y% because Z"
3. Change only ONE element per test
4. Allow 3-7 days for Google to re-crawl
5. Monitor new CTR for 30+ days (exclude first 7 days)
6. Compare at same avg. position; decide: keep, revert, or iterate

### What to Test (Priority Order)

| Priority | Element | Expected Impact |
|----------|---------|-----------------|
| 1 | Add/remove number in title | +15-25% CTR |
| 2 | Add/remove year | +10-15% CTR |
| 3 | Add brackets/parentheses | +10-38% CTR |
| 4 | Change power word | +5-12% CTR |
| 5 | Add CTA in description | +5-10% CTR |

### Interpreting Results

| Scenario | Action |
|----------|--------|
| CTR up, position stable | Keep the change |
| CTR up, position up | Keep and document |
| CTR down, position stable | Revert immediately |
| CTR flat, position stable | Try a bolder variation |

| Position | Desktop | Mobile | Combined |
|----------|---------|--------|----------|
| 1 | 31.7% | 26.9% | 29.3% |
| 2 | 17.5% | 15.1% | 16.3% |
| 3 | 11.3% | 10.2% | 10.8% |
| 4-5 | 5.8-7.9% | 5.2-7.1% | 5.5-7.5% |
| 6-10 | 2.2-4.4% | 1.8-3.9% | 2.0-4.2% |

### When CTR Is Below Benchmark

| Gap | Likely Cause | Action |
|-----|-------------|--------|
| 50%+ below | Title doesn't match intent | Rewrite to align with query intent |
| 25-50% below | Weak title or strong SERP features | Add power words, numbers, brackets |
| 10-25% below | Competitors have better titles | Analyze top 3 and differentiate |

## Multi-Language Optimization

| Language | Max SERP Chars | Key Consideration |
|----------|---------------|-------------------|
| English | 55-60 | Standard baseline |
| Spanish/French/Portuguese | 55-60 | Longer phrases; use abbreviations |
| German | 50-55 | Compound words are long |
| Japanese/Korean | 30-35 | CJK characters ~2x width |
| Chinese | 28-32 | CJK; brackets common |
| Arabic | 50-55 | RTL display; test SERP rendering |

**Principles**: Do not translate literally; do locale-specific keyword research; adjust CTA culturally; for CJK aim 70-80 char descriptions (vs 150-160 Latin).

## Title Tag Modifier Library

| Purpose | Modifiers |
|---------|----------|
| Freshness | 2026, Updated, Latest, This Month |
| Comprehensiveness | Complete, Ultimate, Definitive, In-Depth |
| Ease | Easy, Simple, Quick, Beginner, No-Code |
| Authority | Expert, Pro, Advanced, Data-Driven |
| Value | Free, Budget, Affordable, Open-Source |
| Format | Guide, Checklist, Template, Cheat Sheet, Playbook |
| Social Proof | Trusted, [N]+ Users, Award-Winning, #1 Rated |

**Placement**: Front when modifier IS the keyword ("Free SEO Tool"), parenthetical at end for value signal ("SEO Tools (Free)"), pipe/dash at end for brand.

## Common Title Tag Mistakes

| Mistake | Fix |
|---------|-----|
| Keyword stuffing | Use primary keyword once; vary phrasing |
| Too generic ("Home") | Write a benefit-driven title |
| All caps | Use title case or sentence case |
| Duplicate titles across pages | Unique title per page |
| Brand name first (non-branded) | Put brand at end after pipe |
| Exceeding 60 characters | Front-load important info |
| Misleading clickbait | Title must match content |
