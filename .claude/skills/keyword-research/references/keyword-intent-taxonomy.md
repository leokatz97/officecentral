# Keyword Intent Taxonomy

Classification system for search intent, mapping keywords to content strategies and conversion funnels. **4 primary intent types**, each with 3-4 sub-categories.

---

## 1. Informational Intent

The user wants to **learn something**.

| Sub-category | Example Queries |
|-------------|----------------|
| **Educational** | "what is SEO", "content marketing explained" |
| **Instructional** | "how to set up Google Analytics", "how to do keyword research" |
| **Exploratory** | "types of SEO strategies", "link building techniques" |
| **Troubleshooting** | "why is my site not ranking", "404 error fix" |

**Signal Words**: what, how, why, when, who, where, guide, tutorial, learn, explain, examples, tips, definition, meaning, difference between, pros and cons, checklist, steps

**Content Formats**:

| Sub-category | Best Formats | Target Length |
|-------------|-------------|--------------|
| Educational | Definitive guides, explainers | 1,500-3,000 words |
| Instructional | Step-by-step tutorials | 2,000-4,000 words |
| Exploratory | Listicles, roundups, comparison tables | 2,000-3,500 words |
| Troubleshooting | FAQ pages, debug guides | 800-2,000 words |

---

## 2. Navigational Intent

The user wants to **find a specific website or resource**.

| Sub-category | Example Queries |
|-------------|----------------|
| **Brand Search** | "Ahrefs", "HubSpot CRM" |
| **Product/Feature Search** | "Ahrefs keyword explorer", "GSC coverage report" |
| **Login/Access** | "Ahrefs login", "Google Analytics sign in" |
| **Support/Docs** | "Ahrefs API documentation" |

**Signal Words**: [brand name], login, sign in, pricing, support, help, docs, download, official, website

---

## 3. Commercial Investigation Intent

The user is **researching before a purchase decision**.

| Sub-category | Example Queries |
|-------------|----------------|
| **Comparison** | "Ahrefs vs SEMrush", "WordPress vs Squarespace for SEO" |
| **Review-seeking** | "Ahrefs review 2026", "is Moz worth it" |
| **Best-of Research** | "best SEO tools", "top keyword research tools" |
| **Evaluation** | "SEO tools for small business", "enterprise SEO features" |

**Signal Words**: best, top, vs, versus, comparison, review, alternative, for [audience], pros and cons, worth it, features, pricing, cost, free vs paid

**Content Formats**:

| Sub-category | Best Formats | Target Length |
|-------------|-------------|--------------|
| Comparison | Side-by-side "X vs Y" articles | 2,500-4,000 words |
| Review-seeking | In-depth reviews, case studies | 2,000-3,500 words |
| Best-of Research | Ranked listicles, roundups | 3,000-5,000 words |
| Evaluation | Buying guides, ROI calculators | 2,000-3,500 words |

---

## 4. Transactional Intent

The user wants to **complete a specific action** (purchase, signup, download).

| Sub-category | Example Queries |
|-------------|----------------|
| **Purchase** | "buy Ahrefs subscription", "SEMrush pricing plans" |
| **Signup/Trial** | "Ahrefs free trial", "sign up for Moz" |
| **Download** | "download Screaming Frog", "SEO checklist PDF" |
| **Hire/Engage** | "hire SEO consultant", "SEO agency near me" |

**Signal Words**: buy, purchase, price, discount, coupon, free trial, demo, signup, download, hire, book, schedule, near me, cheap, affordable

---

## Intent Mapping to Conversion Funnel

```
FUNNEL STAGE        INTENT TYPE              CONTENT GOAL
------------------------------------------------------------
Awareness       ->  Informational            Attract & educate
Interest        ->  Informational            Deepen engagement
Consideration   ->  Commercial Investigation Build preference
Evaluation      ->  Commercial Investigation Overcome objections
Decision        ->  Transactional            Convert
Retention       ->  Navigational             Support & retain
```

### Conversion Potential by Intent

| Intent Type | Avg. Conversion Rate | Typical CTA |
|------------|---------------------|-------------|
| Informational | 0.5-2% | Newsletter signup, content download |
| Navigational | N/A (already engaged) | Login, access feature |
| Commercial Investigation | 2-5% | Free trial, demo request |
| Transactional | 5-15% | Buy now, subscribe, hire |

## Mixed Intent Queries

| Query Pattern | Primary Intent | Secondary Intent | Content Strategy |
|--------------|---------------|-----------------|-----------------|
| "best [product] for [use case]" | Commercial | Informational | Buying guide with educational context |
| "how to [task] with [product]" | Instructional | Navigational | Tutorial featuring the product |
| "[product] review and pricing" | Commercial | Transactional | Review with clear pricing CTAs |
| "what is [concept] tools" | Informational | Commercial | Explainer transitioning to tool recommendations |

### Handling Mixed Intent

1. Lead with primary intent -- answer the dominant question first
2. Bridge to secondary intent naturally
3. Match SERP format -- if SERPs show mixed results, your content should too
4. Segment CTAs for different intent segments within the page

---

## Step-by-Step Classification Process

1. Read the query literally -- what is the user asking for?
2. Check for signal words against the tables above
3. Verify against SERP -- what does Google actually show? (ground truth)
4. Identify sub-category from primary intent
5. Check for mixed intent
6. Map to content format based on intent + sub-category
7. Assign funnel stage
8. Set conversion expectation and CTA

### Common Classification Mistakes

| Mistake | Example | Correct Classification |
|---------|---------|----------------------|
| All "best" queries = informational | "best CRM software" | Commercial Investigation |
| All questions = informational | "how much does Ahrefs cost" | Commercial / Transactional |
| Ignoring local intent | "SEO services" | Transactional (local) |
| Missing navigational intent | "HubSpot blog" | Navigational |
| Assuming single intent | "SEO tools" | Mixed -- check SERP |
